# 법적 필수 기재사항 및 툴팁 매핑 (Legal Requirements)

이 문서는 시스템의 분석 결과 화면에서 각 항목별로 표시되는 **법적 근거 및 툴팁 메시지**를 정의합니다.

## 용도 및 기능
- **법적 근거 제공**: 근로계약서 분석 결과 테이블에서 각 항목(예: 임금, 소정근로시간 등)에 마우스를 올리거나 클릭했을 때 나타나는 상세 법령 정보를 제공합니다.
- **카테고리 분류**: 각 항목이 법적으로 '서면 명시 필수'인지, '단순 명시'만 해도 되는지 등을 분류하여 색상과 함께 표시합니다.
- **사용자 가이드**: 비전문가 사용자가 각 항목이 왜 중요한지, 어떤 법적 근거가 있는지 쉽게 이해할 수 있도록 부드러운 말투의 가이드를 제공합니다.

## 카테고리별 색상 시스템
- **서면필수 (`#ef4444`)**: 반드시 서면으로 작성하고 교부해야 하는 가장 중요한 항목 (빨강)
- **서면필수(특수) (`#f59e0b`)**: 기간제나 단시간 근로자에게만 서면 명시가 필수인 항목 (주황)
- **명시필수 (`#3b82f6`)**: 명시는 해야 하나 서면 교부까지는 강제되지 않는 항목 (파랑)
- **필수 (`#8b5cf6`)**: 계약 체결 및 교부 자체에 관한 일반적인 의무 (보라)
